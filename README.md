## Sobre mí

Actualmente soy el fundador y propietario de **ShodTeam**, una organización enfocada en el desarrollo de software, proyectos open source, herramientas para desarrolladores, experiencias para gaming y tecnologías impulsadas por inteligencia artificial.

También lidero el desarrollo de **ShodAgent**, una plataforma de inteligencia artificial **Open Source** diseñada para funcionar con modelos locales, priorizando la privacidad, el rendimiento y la accesibilidad multiplataforma. El proyecto busca ofrecer una experiencia moderna disponible para escritorio, web y consola, impulsada por tecnologías abiertas y una comunidad colaborativa.

## Proyectos

| Proyecto | Descripción |
|---|---|
| [**ShodAgent**](https://github.shodzery.xyz) | Plataforma de inteligencia artificial Open Source enfocada en modelos locales, privacidad, automatización segura y compatibilidad multiplataforma. Permitirá controlar tareas en la PC con permisos seguros, revisar proyectos, organizar archivos, explicar errores, generar documentación y ejecutar acciones siempre con confirmación del usuario. Saldrá al público próximamente. |
| [**ShodVault**](https://github.shodzery.xyz) | Aplicación local para guardar conocimiento, notas y documentos con IA local. Combina ideas de Notion, Obsidian y asistentes inteligentes, permitiendo crear notas en Markdown, organizar proyectos, resumir contenido, generar flashcards, hacer búsquedas semánticas y chatear con tus propios apuntes sin subir información a la nube. |
| [**ShodDesk**](https://github.shodzery.xyz) | Panel de control para proyectos Open Source. Permitirá ver estados, tareas, versiones, commits, issues, roadmap, notas, documentación, builds y releases en un solo lugar. Incluirá IA local para analizar el avance del proyecto y sugerir qué falta por completar. |
| [**ShodModel Manager**](https://github.shodzery.xyz) | Administrador moderno para modelos locales de IA. Permitirá ver modelos instalados, descargar y borrar modelos, consultar peso, RAM recomendada, velocidad, benchmarks locales, tokens por segundo y recomendaciones según CPU, RAM o GPU. También incluirá perfiles como coding, chat, rápido y estudio. |
| [**ShodHub**](https://github.shodzery.xyz) | Centro principal de aplicaciones de ShodTeam. Funcionará como un launcher para instalar, abrir, actualizar y administrar herramientas como ShodChatAI, ShodAgent, ShodVault, ShodModel Manager y ShodDesk desde un solo lugar. Será la base del ecosistema Shodzery. |
| [**ShodChatAI**](https://github.shodzery.xyz) | Aplicación de inteligencia artificial local enfocada en privacidad, modelos locales y uso multiplataforma. Permitirá usar IA desde interfaz web, escritorio o consola, con soporte para modelos locales, configuraciones avanzadas, historial, perfiles y herramientas para estudiantes, programadores y usuarios técnicos. Saldrá al público próximamente. |
| [**Shodzery Launcher**](https://github.shodzery.xyz) | Launcher personalizado para la comunidad. Instalación rápida, diseño moderno, soporte para jugadores premium y no premium, gestión de versiones, mods, shaders y una experiencia optimizada para jugar. Este proyecto tardará más en salir, ya que ShodTeam está desarrollando primero la base del ecosistema Shodzery y sus herramientas principales. |
| [**ShodTeam**](https://github.shodzery.xyz) | Sitio oficial del equipo. Hub central de proyectos, miembros, comunidad, documentación, novedades y enlaces importantes del ecosistema Shodzery. |
